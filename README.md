# react-context-typescript

Typescript flavoured versions based on [Robin Wieruch React Context blog post material](https://www.robinwieruch.de/react-context) (Context, Dynamic Context, UseContext)

- **00 Basic:** Constant context information shared, [codesandbox](https://codesandbox.io/s/new-platform-yz6ls)

- **01 Dynamic Context:** Allow child components to update context information, [codesandbox](https://codesandbox.io/s/determined-elion-4uw8o)

- **02 useContext:** Making easier consuming contet using **useContext** hook, [codesandbox](https://codesandbox.io/s/wispy-water-ku29f)
